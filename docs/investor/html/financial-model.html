Key assumptions

  • Lean team. Founder salary $4.5K in Y1, $0 in Y2, scaling with revenue. First hire in Year 5.
  • Low costs. Pure SaaS — ~$0.18/kit AI cost plus hosting. No inventory.
  • Live product. tailorsend.cc is operating today with Stripe billing — this is not a concept.
  • Upside excluded. Campus site licenses, employer seats, and enterprise API are not in these numbers.
